= Retourner la pièce défectueuse à NetApp - AFX 1K
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Si un composant de votre système de stockage AFX 1K tombe en panne, renvoyez la pièce défectueuse à NetApp. Consultez le  https://mysupport.netapp.com/site/info/rma["Retour de pièces et remplacements"] page pour plus d'informations.
