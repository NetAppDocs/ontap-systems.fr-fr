= Retournez le support de démarrage défaillant à NetApp - FAS500f
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Si un composant de votre système FAS500f tombe en panne, retournez la pièce défectueuse à NetApp. Consultez la  https://mysupport.netapp.com/site/info/rma["Retour de pièces et remplacements"] page pour plus d'informations.
