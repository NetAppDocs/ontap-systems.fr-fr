= Renvoyer le support de démarrage défaillant à NetApp - AFF A800
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Si un composant de votre système de stockage AFF A800 tombe en panne, renvoyez la pièce défectueuse à NetApp. Voir le https://mysupport.netapp.com/site/info/rma["Retour de pièces et remplacements"] page pour plus d'informations.
