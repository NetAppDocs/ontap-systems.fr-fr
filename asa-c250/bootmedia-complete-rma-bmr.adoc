= Retournez le support de démarrage défaillant à NetApp - ASA C250
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Si un composant de votre système ASA C250 tombe en panne, retournez la pièce défectueuse à NetApp. Voir la  https://mysupport.netapp.com/site/info/rma["Retour de pièces et remplacements"] page pour plus d'informations.
