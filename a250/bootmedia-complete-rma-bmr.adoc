= Retournez le support de démarrage défaillant à NetApp - AFF A250
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Si un composant de votre système AFF A250 tombe en panne, retournez la pièce défectueuse à NetApp. Consultez la page  https://mysupport.netapp.com/site/info/rma["Retour de pièces et remplacements"] pour plus d'informations.
